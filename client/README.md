### How to start it up (on Ubuntu):
1. Clone the repo
``git clone git@github.com:M-M-duo/PRIYOMysh.git``
2. Reach the clinet repo:
``cd client``
3. Give permission for runner file:
```chmod +x run.sh```
5. Start run file
```./run.sh```
