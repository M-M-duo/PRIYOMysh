#### Available functions
- Auto-ping server
- Manual server request
- Monitoring server's  json responses
- beta: login/register dialog-window



### How to start it up (on Ubuntu):
1. Clone the repo
```bash
git clone git@github.com:M-M-duo/PRIYOMysh.git
```
3. Reach the clinet repo:
```bash
cd client
```
5. Give permission for runner file:
```bash
chmod +x run.sh
```
7. Start run file
```bash
./run.sh
```
